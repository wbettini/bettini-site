# Bettini Site

This repository powers [www.bettini.us](http://www.bettini.us), the professional portfolio and resume site of Bill Bettini — Data & AI Strategist, Enterprise Architect, and MSDAAI candidate.

## 🔍 Purpose

The site showcases Bill’s:
- Career history and leadership in enterprise-scale data strategy
- Technical skills across AI/ML, automation, analytics, and architecture
- Ongoing academic and professional development
- Open-source projects, including PermitFlow and agentic AI prototypes

## 📁 Structure

Content is modular and stored in the `src/` folder:
- `about.md` — Professional summary and core strengths
- `experience.md` — Career history and roles
- `education.md` — Academic background
- `skills.md` — Technical competencies grouped by domain
- `learning.md` — Courses, certifications, and learning philosophy
- `projects.md` — Featured open-source initiatives

## 🚀 Deployment

The site is deployed via Azure Static Web Apps and auto-updates from GitHub. Future enhancements include:
- Live status badges and CI/CD integration
- Dynamic resume generation from structured data
- Subdomain expansion for project-specific sites (e.g., `permitflow.bettini.us`)
